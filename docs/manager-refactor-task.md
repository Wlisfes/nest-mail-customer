## 技术方案

### 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│ 顶部工具栏（搜索 / 通知 / 主题切换 / 用户菜单+退出登录）      │
├──────────┬──────────────────────────────────────────────────┤
│ 侧边栏    │  内容区（仪表盘 / 收件箱 / 邮件详情 / 写信）       │
│ ─ 写信    │                                                  │
│ ─ 仪表盘  │  仪表盘包含:                                      │
│ ─ 文件夹   │  · 4 个统计卡片                                  │
│   收件箱   │  · 邮件收发趋势折线图（ECharts）                   │
│   已发送   │  · 邮箱来源分布饼图（ECharts）                     │
│   草稿箱   │  · 每周活跃度柱状图（ECharts）                     │
│   星标邮件 │  · 最近邮件预览列表                                │
│   垃圾箱   │                                                  │
│ ─ 邮箱列表 │                                                  │
│ ─ 存储空间 │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

### 图表方案

使用 **ECharts** 实现三种图表：

- **折线图** — 7 天邮件收发趋势，带渐变填充
- **饼图** — 邮箱来源分布（QQ/网易/Gmail/Outlook），环形设计
- **柱状图** — 每周活跃度，带渐变柱体

图表支持暗色模式自动切换，监听 Pinia store 主题变化后重新渲染。

### 退出登录

通过顶部工具栏用户下拉菜单触发，清除 cookie 中的 `APP_NEST_TOKEN`，跳转至 `/main/login`。

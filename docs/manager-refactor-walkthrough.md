## 新增功能详解

### 1. 仪表盘（mail-dashboard.vue）

仪表盘作为登录后的默认视图，提供邮件数据概览：

- **统计卡片**：总邮件数 / 未读邮件 / 今日收到 / 今日发送
    - 每张卡片使用不同颜色渐变背景，hover 时浮起动效
- **邮件收发趋势折线图**（ECharts）
    - 展示最近 7 天的收到/发送趋势
    - 双线 + 渐变面积填充，smooth 曲线
- **邮箱来源分布饼图**（ECharts）
    - 环形饼图展示 QQ/网易/Gmail/Outlook 占比
    - 带圆角间距，鼠标悬浮显示标签
- **每周活跃度柱状图**（ECharts）
    - 渐变色柱体，带圆角顶部
- **最近邮件预览列表**
    - 显示 5 条最近邮件，含发件人头像、主题、时间
- **暗色模式支持**
    - 监听 Pinia store 主题变化，自动销毁并重新渲染 ECharts 实例

> 当前使用模拟数据，后续对接真实后端 API。

### 2. 侧边栏（mail-sidebar.vue）

独立的侧边栏组件，包含：

- **Logo 展示** — Mail Server 品牌标识
- **写信按钮** — 圆角主色调大按钮，禁用态（无邮箱账号时）
- **仪表盘入口** — 📊 图标，高亮当前选中
- **文件夹导航** — 5 个文件夹（收件箱/已发送/草稿箱/星标邮件/垃圾箱）
    - 收件箱带未读数量徽标
    - 选中高亮 + 悬浮背景效果
- **邮箱账号列表**
    - 首字母头像（基于邮箱地址哈希生成渐变色）
    - 邮箱类型标签（QQ/网易/Outlook/Gmail）
    - 悬浮显示删除按钮
- **底部存储空间** — 进度条指示存储使用情况

### 3. 顶部工具栏（mail-toolbar.vue）

- **搜索框** — 圆角无边框输入框，聚焦时显示边框
- **通知铃铛** — 带小红点徽标
- **主题切换** — 太阳/月亮图标，调用 Pinia store 切换暗色模式
- **用户下拉菜单** — 首字母头像，点击弹出：
    - ⚙️ 账号设置
    - 🚪 退出登录（确认弹窗 → 清除 cookie token → 跳转登录页）

### 4. 收件箱优化（mail-inbox.vue）

- **工具栏** — 全选复选框 + 刷新按钮 + 已选中计数 + 总邮件数
- **发件人头像** — 首字母圆形头像，基于地址生成颜色
- **未读指示** — 蓝色小圆点 + 加粗主题文字
- **选中状态** — 复选框 + 浅蓝背景高亮

### 5. 主页面布局（index.vue）

```
视图模式: dashboard | inbox | detail | compose

默认视图 → dashboard（仪表盘）
点击文件夹 → inbox（收件箱列表）
点击邮件  → detail（邮件详情）
点击写信  → compose（写信表单）
```

## 原始需求

```
使用mjml、ckeditor5、vue-draggable-plus做一个模板管理页面，可以根据mjml中的组件拖拽生成模板，后续可以根据模板发送邮件
```

## 澄清问答

### Q1: 模板编辑器交互模式
**A**: 纯拖拽式。左侧组件面板，拖到中间画布排列，右侧属性面板编辑样式。CKEditor 5 用于文本块的富文本编辑。

### Q2: MJML 组件范围
**A**: 以下组件全部支持，后续可能补充：
- `mj-section` — 行/段落容器
- `mj-column` — 列布局（1/2/3列）
- `mj-text` — 文本块（内嵌 CKEditor 5）
- `mj-image` — 图片
- `mj-button` — 按钮
- `mj-divider` — 分隔线
- `mj-social` — 社交链接
- `mj-hero` — 大图横幅

### Q3: 模板管理功能
**A**:
- 模板为**个人私有**
- 需要**分类/分组**（如营销类、通知类等）
- 需要**预设模板**（系统内置模板供快速开始）

### Q4: 模板与发送的关系
**A**:
- 需要**独立的模板发送页面**（选模板 → 填变量 → 选收件人 → 发送）
- 模板中需要支持**变量占位符**（如 `{{姓名}}`、`{{公司}}`）
- 后续会有独立的**模板变量管理页面**

### Q5: 数据存储
**A**: MJML 源码和编译后的 HTML **都需要保存**

### Q6: 优先级与范围
**A**: 一次性完成完整的模板管理系统

## 澄清结论

### 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    邮件模板管理系统                        │
├─────────────┬───────────────┬───────────────────────────┤
│  模板列表页   │  拖拽编辑器页   │  模板发送页               │
│  - 分类筛选   │  - 组件面板     │  - 选择模板               │
│  - CRUD      │  - 画布拖拽     │  - 填充变量               │
│  - 预设模板   │  - 属性面板     │  - 选择收件人             │
│  - 预览/复制  │  - 实时预览     │  - 预览 & 发送            │
├─────────────┴───────────────┴───────────────────────────┤
│                    模板变量管理页                          │
│                  - 定义/管理变量                           │
└─────────────────────────────────────────────────────────┘
```

### 核心页面

| 页面 | 路由 | 说明 |
|------|------|------|
| 模板列表 | `/manager/templates` | 按分类展示模板卡片，支持新建/编辑/删除/复制/预览 |
| 模板编辑器 | `/manager/template-editor/:id?` | 三栏布局拖拽编辑器 |
| 模板发送 | `/manager/template-send/:id` | 基于模板填充变量后发送 |
| 模板变量 | `/manager/template-vars` | 变量 CRUD 管理（后续迭代） |

### 技术栈

| 技术 | 用途 |
|------|------|
| **MJML** | 邮件模板 DSL，编译为兼容各邮件客户端的 HTML |
| **CKEditor 5** | 文本块的富文本编辑（已集成） |
| **vue-draggable-plus** | 组件面板 → 画布的拖拽排序 |
| **Naive UI** | 属性面板表单、模板列表卡片等 UI 组件 |

### 数据模型

```
tb_mail_template
├── keyId (PK)
├── userId (FK)
├── categoryId (FK → tb_mail_template_category)
├── name — 模板名称
├── description — 模板描述
├── mjmlSource — MJML 源码 (MEDIUMTEXT)
├── htmlContent — 编译后 HTML (MEDIUMTEXT)
├── thumbnail — 缩略图 URL
├── isPreset — 是否预设模板
├── createTime / modifyTime

tb_mail_template_category
├── keyId (PK)
├── userId (FK)
├── name — 分类名称
├── sort — 排序号

tb_mail_template_variable（后续迭代）
├── keyId (PK)
├── userId (FK)
├── name — 变量名（如"姓名"）
├── key — 变量键（如"name"）
├── defaultValue — 默认值
```
